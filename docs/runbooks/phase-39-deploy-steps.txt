STEP 2 — CLOUDFLARE R2 (Cesar, Cloudflare dashboard)
----------------------------------------------------
[ ] R2 > Create bucket. Suggested name: stemquest-media
[ ] Bucket > Settings > Public access > enable "r2.dev subdomain".
    WRITE DOWN the host it gives you, looks like: pub-<hash>.r2.dev
    (host only, no https:// prefix)
[ ] R2 > Manage R2 API Tokens > Create API token:
      - Permission: Object Read & Write
      - Scope: ONLY the stemquest-media bucket
    WRITE DOWN: Access Key ID and Secret Access Key (secret shows once)
[ ] Also note your Cloudflare Account ID (right sidebar of the R2 page
    or dashboard home).
    No bucket CORS config needed — media is loaded via <img>/<a> tags,
    not fetch/XHR.
